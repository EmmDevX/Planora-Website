 Planora – Smart Daily Planner App

Planora is a simple and modern **web-based planner application** built with React.  
It helps users organize their daily tasks, stay productive, and manage time efficiently.

 Features

🔐 User authentication (Login / Register)
🏠 Beautiful landing page (Hero, About, How It Works, Pricing)
 📋 Personal task planner dashboard
 ✔️ Add, view, and manage daily tasks
 🔒 Protected routes (only logged-in users can access app)
 🎨 Clean and responsive UI design

 Tech Stack

 ⚛️ React.js
 🎯 React Router DOM
 🎨 CSS3
 💾 LocalStorage (for authentication & tasks)
⚡ Framer Motion (animations)
